  Image Segmentation demo for Avenger96

  Running the demo from the command line with 2 background images:
    # ./image_segmentation_demo.py -b background_1.jpg -b background_2.jpg

  More "-b ..." arguments will add more background images.


  Files in this folder:
 image_segmentation_demo.py               - Python demo code
 lite-model_deeplabv3_1_metadata_2.tflite - TensorFlow Lite image segmentation model
                                          see: https://www.tensorflow.org/lite/examples
 background_1.jpg                         - sample background image for the demo
 background_2.jpg                         - sample background image for the demo
 background_3.jpg                         - sample background image for the demo
 background_4.jpg                         - sample background image for the demo
 background_5.jpg                         - sample background image for the demo
